Abstract

Para gerar páginas de forma numerada é necessário ter em consideração que as variáveis no ambiente XSL - Extensible Stylesheet Language - são imutáveis. Para conseguir associar uma página Html - HyperText Markup Language - a uma entrada //doc associa-se a variável à position() do nó na árvore xml. Esta associação está embrulhada num elemento xml: for-each que permite aplicar o mesmo template a todos os nós selecionados, existindo uma variável "nova" a cada iteração do ciclo com o valor correspondente à posição na árvore.

Para executar o servidor nodejs: Abrir o terminal na diretoria homework e executar o comando node servidor.js&; Posteriormente, assim que o daemon estiver a correr, deve-se indicar o seguinte endereço no browser, localhost:7777/XXX em que XXX corresponde a um valor entre 1 e 448 - este intervalo é o número de páginas Html geradas para o website do exercício;

Esta solução não permite a utilização das funcionalidades "Anterior" e "Seguinte" ou até mesmo o "Índice".
